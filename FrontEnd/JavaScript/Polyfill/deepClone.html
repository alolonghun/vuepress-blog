# JSON.stringify

# 实现原理

调用 JSON.stringify 将对象转换为 JSON 字符串，其序列化对象的顺序如下：

  1. 如果对象存在 toJSON 方法，则调用该方法；
  2. 如果提供了 JSON.stringify 的第二个参数，就应用这个函数过滤器，传入的值为第 1 步返回的值；
  3. 对第 2 步返回的每个值进行序列化；
  4. 如果提供第三个参数，执行相应格式化操作；

# 代码实现

function clone(obj) {
    return JSON.parse(JSON.stringify(obj));
}

# 缺点

  1. 对于数组，属性值为 Undefined、Symbol、Function、Infinity、NaN 都会转换为 null；
  2. 对于对象：
    • 属性值为 Undefined、Symbol、Function 不能被转换，会直接被过滤掉；
    • 属性值为 Infinity、NaN 会被转换为 null；
  3. 属性值为 RegExp、Error 等，会被转换为 {}；
  4. 循环引用会抛出错误；

# 递归实现

function clone(target, map = new WeakMap()) {
    if (typeof target === 'object') {
        let cloneTarget = Array.isArray(target) ? [] : {};

        if (map.get(target)) {
            return map.get(target);
        }
        map.set(target, cloneTarget);

        for (let key in target) {
            cloneTarget[key] = clone(target[key], map);
        }

        return cloneTarget;
    } else {
        return target;
    }
}
