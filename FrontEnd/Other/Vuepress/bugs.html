# vuepress 搭建博客遇到的问题

# 导航栏无法点击展开（移动端）

将 vuepress 升级到 v1.4.1 及以上即可解决：

rm -rf node_modules # 清空依赖
npm i vuepress@latest # 安装最新版本 vuepress

# 运行时无法自动更新

watchpack 依赖版本更新所导致的问题，解决方法： 安装 watchpack@1.6.1 来解决（参见 issue）

npm i -D watchpack@1.6.1
