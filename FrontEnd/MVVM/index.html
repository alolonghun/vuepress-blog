# 前言

MVVM 模型，即由 Model，View，ViewModel 三部分组成；
Model 负责数据管理，View 负责视图，ViewModel 则是 Vue/React 框架内部提供的能力；
像 Vue/React 这些前端框架最大的创新点就是 数据驱动视图，这使得数据交互复杂的应用能更专注于数据的变更，而不用花大量精力去操作 DOM，这样更多更加大型且复杂的应用程序才得以产生；
本章节也是主要记录本人对于 MVVM 框架的认知与理解；
